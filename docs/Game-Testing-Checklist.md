- [ ] Corrupted graphics
- [ ] Wrong colors
- [ ] Lighting issues
- [ ] Shadow problems

### Performance
- [ ] Slow loading

- [ ] Frame drops

### Audio
- [ ] No sound

- [ ] Audio desync
- [ ] Wrong volume

### Controls
- [ ] Unresponsive controls
- [ ] Wrong button mapping
- [ ] Controller not detected

### Saving
- [ ] Can't save
- [ ] Save corruption
- [ ] Load failure
- [ ] Auto-save issues

### Crashes
- [ ] Crashes on launch
- [ ] Crashes in-game
- [ ] Crashes on save
- [ ] Random crashes

## Testing Tips

1. **Test at least 10 minutes** per game to catch issues
2. **Test multiple scenes** (menus, gameplay, cutscenes)
3. **Note device temperature** (emulation can get hot)
4. **Monitor battery usage**
5. **Check storage space** before and after
