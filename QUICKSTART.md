# MeloCafe iOS - Quick Start Guide

## เริ่มต้นอย่างรวดเร็ว (Thai)

### 1. ดาวน์โหลด IPA
ไปที่: https://github.com/jk0965844931-netizen/MeloCafe/actions
คลิก build ล่าสุด → ดาวน์โหลด `MeloCafe-iOS-arm64-unsigned.zip`

### 2. ติดตั้ง LiveContainer
- เปิด AltStore
- ค้นหา "LiveContainer"
- กดติดตั้ง

### 3. นำเข้า IPA
- เปิด LiveContainer
- กด "+" → Import from Files
- เลือกไฟล์ MeloCafe.ipa
- รอจนถึงเสร็จ

### 4. เล่นเกม
- เปิด MeloCafe
- คัดลอกไฟล์เกม (.wud, .wux) ไปที่ Documents
- เปิดเกมใน emulator

## Quick Start (English)

### 1. Download IPA
Go to: https://github.com/jk0965844931-netizen/MeloCafe/actions
Click latest build → Download `MeloCafe-iOS-arm64-unsigned.zip`

### 2. Install LiveContainer
- Open AltStore
- Search "LiveContainer"
- Install

### 3. Import IPA
- Open LiveContainer
- Tap "+" → Import from Files
- Select MeloCafe.ipa
- Wait for installation

### 4. Play Games
- Open MeloCafe
- Copy game files (.wud, .wux) to Documents folder
- Load game in emulator

## Minimum Requirements

- iPhone 8 or newer
- iOS 14.0+
- 3GB RAM
- LiveContainer (for JIT)
